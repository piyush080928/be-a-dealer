# React Buy-Sell Platform

A responsive **Buy & Sell web application** built with **React** and **Tailwind CSS**.  
This project includes advanced React features like routing, form validation, notifications, and animations.

---

## Features

- User Authentication (Login / Signup) with **React Hook Form** and **Yup**  
- Dashboard to view balance and purchased products  
- Buy Products with smart negotiation system  
- Sell Products with bargain mode simulation  
- Responsive Navbar & Sidebar for mobile and desktop  
- Product Cards with images and descriptions  
- Notifications using **react-hot-toast**  
- Smooth animations using **Framer Motion**

---

## Technologies Used

- React  
- React Router DOM  
- React Hook Form & Yup  
- React Icons  
- React Hot Toast  
- Tailwind CSS  
- Framer Motion
